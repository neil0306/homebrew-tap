# neil0306/homebrew-tap

Homebrew tap for [neil0306](https://github.com/neil0306)'s tools.

## Install

```bash
brew install neil0306/tap/bltusb
```

or

```bash
brew tap neil0306/tap
brew install bltusb
```

## Formulae

- **[bltusb](https://github.com/neil0306/bltusb)** — read/write BitLocker-encrypted USB drives on macOS (Apple Silicon), no macFUSE / no kernel extension.

  After install, run `bltusb install` once to set up the anylinuxfs backend, then just run `bltusb`.
